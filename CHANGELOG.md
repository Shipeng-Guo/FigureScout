# Changelog

所有重要的变更都将记录在这个文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2025-11-10

### 新增功能
- ✨ 精确到天的日期显示（YYYY-MM-DD格式）
- ✨ 杂志名称显示功能
- ✨ 智能排序功能（相关性/时间/杂志）
- ✨ 实时进度条和状态显示
- ✨ 三阶段搜索进度可视化

### 修复
- 🐛 修复 PMC 链接重复前缀问题（PMCPMC → PMC）
- 🐛 修复时间过滤器无效问题（动态调整 page_size）
- 🐛 修复杂志名称不显示问题（支持嵌套JSON结构）
- 🐛 修复浏览器缓存导致的前端更新问题

### 改进
- 🎨 优化搜索结果展示界面
- 🎨 改进日期和杂志信息布局
- ⚡ 动态调整API返回数量（基于时间范围）
- 📝 整合所有文档为单一README.md

### 技术优化
- 🔧 更新 `.gitignore` 配置
- 🔧 精简项目结构（源代码 280KB）
- 🔧 删除临时文件和缓存
- 🔧 优化Git仓库大小（< 500KB）

## [1.0.0] - 2025-11-10

### 首次发布

#### 核心功能
- 🔍 Europe PMC 全文搜索集成
- 📄 PubMed Central (PMC) 全文内容提取
- 🎯 关键词高亮显示
- 📊 相关性评分系统
- 🖼️ 图表和图注提取

#### 技术栈
- **后端**: Flask + Python 3.8+
- **前端**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS
- **API**: Europe PMC API + PubMed E-utilities

#### 支持的功能
- 高质量期刊过滤（Nature系列、Cancer系列等）
- 开放获取文献搜索
- 全文章节提取（方法、结果、讨论）
- 响应式UI设计
